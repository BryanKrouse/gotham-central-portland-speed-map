# Portland Ledger - Traffic Speed Observations

An interactive web dashboard for exploring Portland traffic speed observation data through a dark, municipal-inspired cartographic interface. The dashboard visualizes posted speed limits, the percentage of vehicles exceeding the posted speed, and the percentage exceeding the limit by 10 mph or more — rendered through an interactive canvas map, coordinated histogram, filtering controls, and statistical summaries.

The interface applies the **Gotham Central** cartographic theme: a high-contrast, ink-dark visual language modeled after internal municipal operations tools, emphasizing restrained color, high information density, and an evidence-led layout that puts the data first. A side panel surfaces variable definitions, class-break legends, and ranked observations alongside the map.

## Design Inspiration

The Gotham Central theme and its cartographic visual language were developed using structured design specifications inspired by the prompt cartography framework of Ian Muehlenhaus — a methodology for translating map design intent into consistent, reproducible outputs through explicit written prompts.

## Built With

Vanilla HTML, CSS, and JavaScript — canvas-rendered map with procedural cartography, SVG histogram, and no external mapping or charting dependencies.
